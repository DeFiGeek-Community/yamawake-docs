# Yamawake

## Yamawakeとは何ですか？

パーミッションレスに（**管理者の許可なく**）バルクオークションの開催、参加が行えるDappです。

## バルクオークションとは何ですか？



## オークション主催者側

設定項目を入力するだけで、誰でも任意のトークンをバルクオークション形式で販売できるノーコードツールとして使用できます。

## ユーザー（入札者）側

任意のトークンのバルクオークション開催中に入札ができ、オークション終了後に割合に応じてトークンを取得できます。
